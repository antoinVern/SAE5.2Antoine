# 📦 Commandes Git

## Première utilisation

```bash
git init
git add .
git commit -m "Initial commit: Laboratoire éphémère SAE5.2"
git remote add origin https://github.com/VOTRE-USERNAME/VOTRE-REPO.git
git branch -M main
git push -u origin main
```

## Commandes courantes

```bash
git add .
git commit -m "Description"
git push
```

## Créer le dépôt sur GitHub

1. Aller sur https://github.com → "+" → "New repository"
2. Donner un nom (ex: "SAE5.2-Laboratoire-Ephemere")
3. **NE PAS** cocher "Initialize with README"
4. Utiliser les commandes ci-dessus
